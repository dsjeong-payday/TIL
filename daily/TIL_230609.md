# 웹 개발
## HTTP
- HyperText Transfer Protocol

### 주요 메소드
#### GET
#### POST
- 주로 HTML form을 통해 전달 -> 서버에 변경사항
- 헤더 `Content-type`: 요청 본문의 유형 특정
    - `text/plain`
    - `application/json` 등
- 헤더 `Authorization`: `POST` 요청을 보내는 주체의 권한 소유 여부 판별

## json
- JSON (JavaScript Object Notation)
- 데이터 형식
- 여러 웹 기반 서비스에서 데이터를 주고 받을 때 활용
